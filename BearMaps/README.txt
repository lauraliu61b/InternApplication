在本地运行MapServer.java，在浏览器中打开http://localhost:4567/map.html查看实现功能。
该程序使用的library在此文件夹上层的javalib中。

Build up a zoom-in, zoom-out and route-searchable map like Google map which can tell the shortest route between two spots in Berkeley based on A* by using the real world, XML-formatted data as input. This is a group project in Berkeley CS61B. My contribution is dealing with the data and storing it as a graph, and implement A* algorithm at the back end.

Language: Java.
Tool: Intellij.
Type: Web Application.
